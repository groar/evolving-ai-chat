- Reasoned behavior:
  - For rows without `improvement_class`, behavior is unchanged (falls back to `"settings-trust-microcopy-v1"`).
  - For rows with `improvement_class`, behavior now correctly respects the stored value, including `system-prompt-persona-v1`, which is already covered by existing tests in `test_proposals.py`.
