# Ticket Metadata

This folder stores supporting metadata, not active ticket states.

- `epics/` optional grouping files (`E-*.md`) that link related tickets.
- `feedback/` PM feedback intake (`inbox/`), catalog (`INDEX.md`), and checkpoint summaries.
- `qa/` periodic QA run reports and finding summaries.
- `templates/` canonical templates for creating tickets.

Do not move ticket files (`T-*.md`) into this folder.
